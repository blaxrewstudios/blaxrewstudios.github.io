A software related boardgame (Multiplayer, Singleplayer, Realife boardgame). This means this game was originally designed a boardgame but the player's stats calculation got to complicated after a while. That is why we wrote a software for our game. If you buy the boardgame the software is normally included.


The main software is already released. You can get access to the software on request. This is just an ALPHA-Release that means you may experience bugs or issues.


Our plans to that game are to make a online multiplayer which will cost for those who did not buy the boardgame. A player versus bot mode is planned to. We want to add a bug report section in the software too.
